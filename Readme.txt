Program execution Instruction
-----------------------------------------------------------------------
Program can be executed running .exe file in 
Lab4\bin\Debug\Lab4.exe
Program execution report will be saved in
Lab4\bin\Debug\report.txt
Please note that data will be only flushed at the end of execution 
of program.
------------------------------------------------------------------------
This program had been writen in C#. Project can be opened by
Visual Studio software and run as well.
------------------------------------------------------------------------
There are 3 seperate .cs files for different execution type algorithm 
implementatioin
1) serial program --> Serial.cs
2) Parallel program --> ParallelClass.cs
3) Optimized parallel program --> OptimizeParallel.cs
